Projeto da Sprint 5, com o intuito de criar um aplicativo web para a consulta de preços de carros.
O aplicativo web oferece quatro funcionalidades diferentes, uma que mostra todos os dados do DataFrame, um histograma de relação Preço x Modelo, um gráfico que oferece a comparação de diferentes dados sobre os carros e uma comparação de preços por modelo.
